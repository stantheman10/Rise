## RISE — Stand Reminder & Habit Tracker

RISE is a minimalist, gamified stand reminder app designed to help desk workers stay active and build healthier habits throughout the day.

Built with React Native and Expo, the app focuses on simplicity, consistency, and subtle motivation through streaks, goals, and achievements.

### ✨ Features

- ⏱ Smart stand reminders (custom frequency)
- 🎯 Daily goal tracking
- 🔥 Streak system to build consistency
- 📊 Activity history with visual insights
- 🏆 Badge & achievement system
- 🎨 Clean, modern dark UI with smooth animations
- 📱 Optimized for Android with safe-area handling

### 🛠 Tech Stack

- React Native (Expo)
- Zustand (state management)
- React Navigation
- Reanimated (animations)
- AsyncStorage (persistence)
- Expo Notifications

### 📌 Purpose

This project was built to explore:
- mobile UX design for habit-forming apps
- state management with persistence
- notification-based behavior loops
- building a production-ready React Native app
